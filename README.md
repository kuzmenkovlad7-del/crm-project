# CRM Project Source Audit

This repository contains a sanitized source snapshot of the existing WordPress-based CRM.

Included:
- wp-content/themes/romance-crm
- custom CRM-related plugins if present

Excluded:
- wp-config.php
- database dumps
- backups
- uploads
- cookies
- secrets
- credentials
- WordPress core

Goal:
Analyze the existing CRM and implement a safe first-stage integration/module for Dating.com according to the signed scope.

Important:
No captcha bypass, anti-bot bypass, scraping protection bypass, spam, or unauthorized automation.
